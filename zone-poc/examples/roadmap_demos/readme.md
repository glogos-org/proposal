# Glogos Roadmap Demos

POC implementations for the Glogos Protocol roadmap vision.

---

## Demos

| Version | Demo                    | Nobel Basis           | Description           |
| ------- | ----------------------- | --------------------- | --------------------- |
| v2.0    | `fishery_demo.py`       | Ostrom 2009           | Commons governance    |
| v3.0    | `employment_demo.py`    | Hart & Holmström 2016 | Contract verification |
| v4.0    | `carbon_credit_demo.py` | Milgrom & Wilson 2020 | Commit-reveal auction |

---

## Quick Start

```bash
cd zone-poc
export PYTHONIOENCODING=utf-8

# Commons Governance (Ostrom)
python examples/roadmap_demos/fishery_demo.py

# Contract Theory (Hart/Holmström)
python examples/roadmap_demos/employment_demo.py

# Market Design (Milgrom/Wilson)
python examples/roadmap_demos/carbon_credit_demo.py
```
